# StayLocal | Premium PG Network 🏠

A high-performance, SEO-optimized PG discovery platform built with **Next.js 15**, **Tailwind CSS**, and **Supabase**.

## 🚀 Key Features
- **Premium UI:** Glassmorphic design with Framer Motion animations.
- **Dynamic Maps:** Precise Google Maps integration for property verification.
- **Real-time Data:** Live PG listings managed via Supabase PostgreSQL.
- **SEO Engine:** Automated JSON-LD schema for 3x organic growth.

## 🛠️ Tech Stack
- **Frontend:** Next.js (App Router), TypeScript
- **Backend:** Supabase (Auth & Database)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion

## 📍 Coverage
Active in Bangalore, Chennai, Hyderabad, and Pune.
